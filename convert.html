위의 kugak.json 다운로드 버튼을 눌러 파일을 저장하세요.
저장한 파일을 GitHub 저장소의 기존 kugak.json과 교체하면 반영됩니다.
사용 방법
1CSV 파일을 위 영역에 드래그하거나 클릭해서 선택합니다.
2구분자와 인코딩을 확인합니다. (기본값: 탭·UTF-8 BOM)
3빈 필드 옵션을 선택합니다. (기본: 생략 — 파일 크기 감소)
4JSON 변환 시작을 클릭합니다.
5kugak.json 다운로드로 파일을 받아 GitHub에 업로드합니다.
